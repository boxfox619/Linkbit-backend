# Linkbit - Backend
암호화폐 간편 송금, 통합 서비스 Linkbit.
복잡하고 긴 암호화폐 지갑 주소를 단축시키고 다양한 암호화폐를 한개의 앱으로 관리하고 송금할 수 있도록 스마트폰 어플리케이션 서비스.

# Tech Stacks
- Vert.x
- Jooq
- Postgresql
- Redis
- JWT
- Firebase
- Web3j
- [Vertx-wetm-library](https://github.com/boxfox619/vertx-wetm-library)
- RxJava
